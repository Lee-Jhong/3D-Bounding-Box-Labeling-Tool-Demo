Installing and Running the Software:

If this is your first time using this software:
Double click install.cmd, and follow the instructions. After installation, it will automatically start the program for you.

Next time when you want to use the tool, double click run.cmd

Note, if on Windows 10, it will give a warning when you try to open the files. Just click "more info" and open anyway.
